# בדיקת זמנים - הפעלה עצמאית

זו אפליקציית React רגילה. כל הנתונים נשמרים מקומית בדפדפן (localStorage) - אין
מסד נתונים חיצוני ואין צורך בהרשמה לשום שירות מלבד GitHub ו-Vercel.

---

## שלב 1: העלאה ל-GitHub

1. כנס ל-**github.com**, צור חשבון אם אין לך (חינם)
2. "New repository" - תן שם, השאר Private אם תרצה
3. בטרמינל/מחשב, בתוך תיקיית הפרויקט:
   ```
   git init
   git add .
   git commit -m "Check timer app"
   git branch -M main
   git remote add origin https://github.com/USERNAME/REPO-NAME.git
   git push -u origin main
   ```
   (אם אין לך git מותקן, אפשר גם לגרור את כל הקבצים דרך ממשק GitHub באתר - "uploading an existing file")

---

## שלב 2: פרסום (Vercel) - כ-3 דקות

1. כנס ל-**vercel.com**, לחץ "Sign Up", והתחבר **עם GitHub** (הכי פשוט)
2. "Add New" → "Project"
3. תבחר את ה-repository שיצרת
4. לחץ **Deploy** (אין צורך בשום Environment Variable)
5. תוך דקה תקבל קישור אמיתי - זה הקישור הקבוע והעצמאי שלך

---

## מה קורה בעתיד

- כל שינוי עתידי בקוד: `git push` - Vercel מפרסם מחדש אוטומטית תוך דקה
- אין תחזוקה שוטפת, אין עלות - Vercel חינמי בהיקף שימוש אישי בלי בעיה

## הוספה למסך הבית בנייד

בספארי/כרום בנייד: שיתוף → הוסף למסך הבית. האפליקציה תיראה ותתפקד כמו אפליקציה
רגילה, ותשמור את הנתונים על המכשיר עצמו.

## הערה חשובה על תזכורות

התזכורות פועלות באמצעות התראות דפדפן (Notification API) בזמן שהאפליקציה/הטאב
פתוחים. אם הדפדפן נסגר לגמרי או שהמכשיר במצב חיסכון בסוללה לאורך זמן, התזכורת
עלולה לא להגיע - זה כרוך בהגבלות הדפדפן ולא באפליקציה עצמה. אם בעתיד תרצה
תזכורות אמיתיות ברקע (push), זה אפשרי אך דורש שרת/מסד נתונים (כמו שיש
ב-Afterglow) - אפשר להוסיף את זה בנפרד אם תרצה.
